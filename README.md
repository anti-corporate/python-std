Here is a example of how the standard library works:

import std;

std.start_() <-- Entry Point

std.class_("Main") <-- Classes (like java)

std.window() <-- dont know what this does

std.fn("main")(std.cout(std.random_(1, 30))) <-- function parameter and print

std.end_() <-- the end

always use " import std; " and std as a prefix.
